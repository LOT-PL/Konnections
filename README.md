 A port of a popular game by the NYT 'Connections' to the kindle! 

## Minimum Requirements 

 - AdBreak, WinterBreak, or LanguageBreak
 - Latest Universal Hotfix
 - Minimum Version of  5.12.2.2

## Installation

1. Grab the latest file from releases
2. Unzip it, paste both `Konnections/` and `Konnections.sh` into the `documents/` folder on your Kindle.
3. Run the scriptlet, and have fun!

## Features
- Shuffle Button
- Stats about the current game
- New game if you dont like the current one
